# 7th Bhava

## Basic Information

- **Primary Quality**: <span className="hindi">वृद्धि</span> (Vriddhi) - Growth, expansion
- **Sanskrit Name**: <span className="hindi">कलत्र भाव</span> (Spouse House)

- **Classification**: <span className="hindi">केंद्र</span> (Kendra/Angular), <span className="hindi">मारक</span> (Marak)
- **Note**: It is 180 degree to <span className="hindi">लग्न</span> (Lagn)<br/><span className="hindi">लग्न</span> = <span className="hindi">आत्मा</span> (Aatma), soul, self<br/>Someone/something exactly opposite to you<br/><span className="hindi">सूर्य अस्त</span> (Surya astha) here - Important planet<br/>If bad brings <span className="hindi">क्लेश</span> (Kalesha)<br/>If marriage is bad then we also feel that we are dead<br/>This house has to be treated with great respect
