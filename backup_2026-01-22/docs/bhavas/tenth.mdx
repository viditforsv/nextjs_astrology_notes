# 10th Bhava

## Basic Information

- **Primary Quality**: <span className="hindi">सिद्धि</span> (Siddhi) - Accomplishment, perfection
- **Sanskrit Name**: <span className="hindi">कर्म भाव</span> (Karma House)
- **Also Known As**: <span className="hindi">कर्म स्थान</span> (Karma Sthan)
- **Significator**: <span className="hindi">बुध</span> (Mercury), <span className="hindi">सूर्य</span> (Sun), <span className="hindi">शनि</span> (Saturn)
- **Classification**: <span className="hindi">केंद्र</span> (Kendra/Angular)
- **Note**: Most important after <span className="hindi">लग्न</span> (Lagn)<br/>Kis field main job hai wo 6th house, kitna rise karoge wo 10th house<br/>No progress without <span className="hindi">सिद्धि</span> (Siddhi)
