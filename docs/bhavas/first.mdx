# 1st Bhava (<span className="hindi">लग्न</span> / Ascendant)

## Basic Information

- **Also Known As**: <span className="hindi">प्रथम भाव</span> (Pratham Bhava) - First house
- **Primary Quality**: <span className="hindi">बुद्धि</span> (Buddhi) - Intellect, discrimination
- **Significator**: <span className="hindi">सूर्य</span> (Sun)
- **Classification**: <span className="hindi">केंद्र</span> (Kendra/Angular), <span className="hindi">त्रिकोण</span> (Trikona/Trinal)
- **Note**: Most important house in the chart
