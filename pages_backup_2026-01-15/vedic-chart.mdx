## Reusable Chart Component

Here's the reusable `VedicChart` component in action:

### Example 1: Jupiter in 4th House

<VedicChart 
  planets={[
    { name: 'Jupiter', house: 4 }
  ]}
  showLabels={true}
  size={400}
/>

### Example 2: Multiple Planets

<VedicChart 
  planets={[
    { name: 'Sun', house: 1 },
    { name: 'Moon', house: 7 },
    { name: 'Mars', house: 10 },
    { name: 'Jupiter', house: 4 }
  ]}
  showLabels={true}
  size={400}
/>

### Example 3: Without Labels

<VedicChart 
  planets={[
    { name: 'Venus', house: 2 },
    { name: 'Mercury', house: 5 },
    { name: 'Saturn', house: 8 }
  ]}
  showLabels={false}
  size={300}
/>
