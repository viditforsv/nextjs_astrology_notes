# 12th Bhava

## Basic Information

- **Sanskrit Name**: <span className="hindi">व्यय भाव</span> (Expenditure House)
- **Significator**: <span className="hindi">शनि</span> (Saturn), <span className="hindi">केतु</span> (Ketu)
