2. **Asteya (Non-stealing)** - Don't use investigative skills to steal/deceive
3. **Tapas (Discipline)** - Channel intensity into constructive work
4. **Ishvara Pranidhana (Surrender)** - Surrender control and manipulation
5. **Brahmacharya (Moderation)** - Balance passionate intensity
