# Sun in Different Houses - SPEIM Analysis

This section applies the SPEIM-Yamas-Niyamas framework to analyze Sun's placement in different houses and rashis. Each placement is examined through all five dimensions to provide comprehensive insights.

## Framework Application

Each analysis includes:
- **SPEIM Dimensions**: Spiritual, Physical, Emotional, Intellectual, Material
- **Niyama Qualities**: Observances for each dimension
- **Yama Qualities**: Restraints for each dimension
- **Three States**: Neutral, Positive (Shubh), and Negative manifestations

## Example: Sun in 1st House (Mesh Rashi)

### Basic Configuration
- **Sun** in **1st House** (Lagna)
- **Rashi:** Mesh (Aries)
- **Lagna Lord:** Mars 
- Sun exalted in Mesh
- 5th Lord in the first house

### SPEIM Framework Analysis

|------------------|--------------|--------------|--------------|--------------|------------|
| **Primary Niyama** | Ishvara Pranidhana | Saucha | Santosha | Svadhyaya | Tapas |

| **Primary Yama** | Asteya | Ahimsa | Brahmacharya | Satya | Aparigraha |

|------------------|--------------|--------------|--------------|--------------|------------|

### Key Observations

**Strengths:**
- Natural leadership and authority
- Strong vitality and presence
- Straightforward and honest nature

**Challenges:**
- Ego and pride
- Dominating behavior
- Excessive need for recognition

**Success Factors:**
- Cultivate humility alongside leadership
- Channel authority toward service
- Balance personal power with spiritual surrender
