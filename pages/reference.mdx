# Quick Reference Notes

Concise notes and reminders for various astrological concepts.

## Dasha Analysis Points

जो ग्रह जिस घर में बैठा है (Jo graha jis ghar mein baitha hai):

1. **जिस के साथ बैठा है** - Which planets it sits with
2. **जिस घर में बैठा है** - Which house it occupies
3. **जिन स्थानों को देखता है** - Which places it aspects
4. **जिन ग्रहों के द्वारा देखा जाता है** - By which planets it is aspected
5. **जिस ग्रह के नक्षत्र में बैठा है** - In whose nakshatra it is placed
6. **जिसके नक्षत्रों के घर में बैठा है, वो मालिक जिस का मालिक है** - Whose nakshatra's house it occupies, that lord whose lord

## Key Terminology

### Sanskrit/Hindi Terms

- **Graha (ग्रह)** - Planet
- **Rashi (राशि)** - Zodiac Sign
- **Bhava (भाव)** - House
- **Nakshatra (नक्षत्र)** - Lunar Mansion
- **Dasha (दशा)** - Planetary Period
- **Yoga (योग)** - Planetary Combination
- **Lagna (लग्न)** - Ascendant
- **Baithak (बैठक)** - Placement/Sitting
- **Dristi (दृष्टि)** - Aspect/Gaze

### Analysis Terms

- **Shubh (शुभ)** - Auspicious/Benefic
- **Ashubh (अशुभ)** - Inauspicious/Malefic
- **Vakri (वक्री)** - Retrograde
- **Uchcha (उच्च)** - Exalted
- **Neecha (नीच)** - Debilitated
- **Moolatrikona (मूलत्रिकोण)** - Primary Dignity
- **Chestha Bal (चेष्ठा बल)** - Motional Strength
